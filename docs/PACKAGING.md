# Упаковка в нативное приложение (опционально)

Тренажёр — статическое веб-приложение (HTML, CSS, ES-модули). Для установки на устройство без браузера можно использовать:

## Capacitor (рекомендуется)

1. Установите [Node.js](https://nodejs.org/) и в корне проекта выполните `npm init -y`.
2. `npm install @capacitor/core @capacitor/cli @capacitor/android @capacitor/ios`
3. `npx cap init` — укажите `webDir` на корень (где лежит `index.html`) или на подпапку сборки.
4. Скопируйте файлы приложения в `webDir`, затем `npx cap add android` / `ios` и откройте проект в Android Studio / Xcode.

Сборка веба не обязательна: можно указывать текущую папку, если все пути относительные (как сейчас).

## Cordova

Альтернатива: [Apache Cordova](https://cordova.apache.org/) с плагином `cordova-plugin-whitelist` и оболочкой `WebView`. После стабилизации функций в браузере имеет смысл добавить оболочку отдельной веткой или репозиторием.

## Ionic / Framework7

Имеет смысл только при **переписывании UI** на компоненты фреймворка. Текущая логика (`lesson-engine.js`, контент модулей) может остаться; перенос — отдельный крупный этап после стабилизации веб-версии.
